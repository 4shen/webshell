# ImgURL
ImgURL是一款简单、纯粹的图床程序，使用PHP + SQLite 3开发。

> 2019-07-21:ImgURL Pro专业版已经发布，支持多个外部存储，MySQL支持、无广告，专业技术支持，自建图床的极佳选择，6折促销中:[https://dwz.ovh/9](https://dwz.ovh/9)

### 主要功能
- [x] 支持拽拖上传、多图上传、Ctrl + V粘贴上传、URL上传
- [x] 支持图片裁剪，自动生成缩略图
- [x] 限制访客上传数量
- [x] 图片压缩/批量压缩
- [x] 图片鉴黄/批量鉴黄
- [x] API支持
- [ ] 在线更新
- [ ] 外部存储

### 环境要求
* PHP >= 5.6
* PDO_SQLite
* GD2
* ImageMagick
* fileinfo
* pathinfo
