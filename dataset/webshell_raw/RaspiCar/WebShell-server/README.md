#WebShell server-end

A simple webshell for embedded systems 

Checkout `webui` branch for UI.

###Build

```
make all
```

###Run

Use whatever CGI/1.1 compatible servers you like (eg. Boa, Apache).

Put these binaries in CGI directory.

Put webui files in DocumentRoot
